The Linux Filesystem


/boot: contains files for starting the system

/dev: contains device files

/etc: contains most system-wide configuration files

/home: users personal directories

/lib: where libraries live. They contain snippets of code that applications use to draw windows on your desktop, control peripherals, or send files to your hard disk.

/media: where external storage is automatically mounted when plugged in

/mnt: where you would manually mount storage devices or partitions

/opt: where software you compile sometimes lands. applications will be in /opt/bin and libraries will be in /opt/lib

/proc: contains information about your computer (cpu, linux kernel, etc.)

/root: home directory of superuser

/run: new directory where system processes use to store temporary data

/sbin: contains applications that only the superuser can use

/srv: contains data for servers

/tmp: contains temporary files by applications you are currently using

/var: contains log files located in /var/log
