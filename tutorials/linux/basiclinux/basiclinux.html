linux command cheatsheet


ls: lists files/directories

ls -al: lists files/directories along with permissions

cd: changes directory

apt-get install package_name: installs program

sudo: gives root permissions to command

cat: allows you to view contents of a file

nano file_name: opens a file with text editor

mkdir directory_name: makes directory

touch file_name: makes file

ip a: shows network information

dhclient: grabs IP address from DHCP server

man command_name: manual for given command

vi file_name: opens a file with a painful text editor
